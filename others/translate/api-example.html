翻译API使用示例

这个页面展示了如何使用我们的翻译API进行文本翻译。本实时翻译工具支持百度翻译API和有道翻译API。

注意： 在使用翻译API之前，请先获取相应的API密钥。

使用方法

1 导入API模块： 
import TranslateAPI from './api.js';
2 配置API密钥： 
// 设置使用的API服务
TranslateAPI.setService('baidu'); // 或 'youdao'

// 设置百度翻译API密钥
TranslateAPI.setApiKey('baidu', 'YOUR_APP_ID', 'YOUR_SECRET_KEY');

// 设置有道翻译API密钥
TranslateAPI.setApiKey('youdao', 'YOUR_APP_ID', 'YOUR_SECRET_KEY');
3 调用翻译方法： 
// 使用异步方式调用翻译API
async function translateExample() {
    try {
        // 翻译文本，从英文到中文
        const result = await TranslateAPI.translate('Hello world', 'en', 'zh');
        console.log('翻译结果:', result);
    } catch (error) {
        console.error('翻译失败:', error.message);
    }
}

支持的语言代码

以下是常用的语言代码：

  • zh - 中文
  • en - 英语
  • ja - 日语
  • ko - 韩语
  • fr - 法语
  • de - 德语
  • es - 西班牙语
  • ru - 俄语
  • auto - 自动检测（仅用于源语言）

完整示例

import TranslateAPI from './api.js';

// 配置翻译API
TranslateAPI.setService('baidu');
TranslateAPI.setApiKey('baidu', 'YOUR_APP_ID', 'YOUR_SECRET_KEY');

// 创建翻译函数
async function translateText() {
    const sourceText = document.getElementById('source').value;
    const sourceLang = document.getElementById('from-lang').value;
    const targetLang = document.getElementById('to-lang').value;
    const resultElement = document.getElementById('result');

    resultElement.textContent = '翻译中...';

    try {
        const translatedText = await TranslateAPI.translate(sourceText, sourceLang, targetLang);
        resultElement.textContent = translatedText;
    } catch (error) {
        resultElement.textContent = `翻译失败: ${error.message}`;
    }
}

// 为翻译按钮添加事件监听
document.getElementById('translate-btn').addEventListener('click', translateText);
提示： 在实际应用中，请妥善保管你的API密钥，避免直接在客户端代码中明文存储。在生产环境中，应通过服务器端进行API调用。

错误处理

翻译API可能会返回以下错误：

  • API密钥未配置或无效
  • 网络连接问题
  • API调用超过限额
  • 不支持的语言组合

请务必在代码中添加适当的错误处理逻辑。
